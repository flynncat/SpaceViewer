# 迭代文档（v1 -> v2）

## v1（已实现）

- 前后端分离结构：`client` + `server` + `shared/data`
- 首页叙事区：强调“看见轨道而非只看参数”
- 3D 预览区：太阳、地球轨道、实时近地体/著名陨石目标轨道
- 目标交互：点击目标切换“选中对象”信息窗口
- 资料库：著名陨石图文卡片 + 关键词搜索 + 类型筛选
- 实时接口：NASA NEO Feed 当日数据拉取与缓存
- API 层：健康检查、著名陨石、实时 NEO、组合首页数据

## v1.5（已实现）

- 时间轴：支持 -3 到 +3 天偏移查询与自动回放
- 三对象对比：同屏比较距离、速度、大小、风险级别
- 收藏能力：本地浏览器持久化收藏列表
- 订阅能力：本地阈值订阅（按最近距离命中高亮）
- 交互增强：收藏、订阅、对比均支持定位与移除

## v1.6（已实现）

- 3D 可视性增强：目标尺寸放大，选中对象增加脉冲位置标记
- 轨道可读性增强：选中轨道高亮，轨迹点悬浮显示时间（UTC）
- 动态运动增强：选中对象显示速度向量箭头，辅助理解飞行方向
- 数据源可观测性：新增源状态灯（Feed/Browse/本地图鉴/本地兜底）
- 卡片体验升级：随机图文卡片 + 点击弹窗 + 搜索/视频/专业站分组外链

## v2（建议）

- 通知订阅：关注对象接近阈值提醒
- 大屏模式：展馆演示布局
- 多语言内容与教育课程模式
- AI 自动生成对象科普解读卡片
